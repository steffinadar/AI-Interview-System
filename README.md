# AI Interview Preparation & Resume Analyzer

An AI-powered Interview Preparation and Resume Analysis System developed using Python, Flask, Machine Learning, and NLP techniques. This project helps students improve placement preparation by analyzing resumes, calculating ATS scores, extracting technical skills, predicting job roles, and generating AI-based interview questions.

# Features

* ATS Resume Score Prediction
* Resume PDF Upload & Analysis
* Skill Extraction using NLP
* Job Role Prediction
* Resume Improvement Suggestions
* AI-Based Interview Questions
* Interactive Dashboard UI
* Mock Interview Preparation
* Responsive Gradient User Interface

# Technologies Used

* Python

* HTML5

* Scikit-learn

# Project Modules

### Resume Analyzer

Analyzes uploaded resumes and extracts technical skills, keywords, and ATS scores.

## Job Role Prediction

Predicts suitable job roles such as:

* Data Scientist
* Backend Developer
* Frontend Developer
* Cloud Engineer

## AI Interview Preparation

Generates technical and HR interview questions based on extracted skills.

## Resume Suggestions

Provides improvement recommendations to enhance resume quality and placement chances.

# How It Works

1. User logs into the system
2. Uploads resume in PDF format
3. System extracts resume text
4. NLP algorithms detect skills and keywords
5. ATS score is calculated
6. Job role is predicted
7. AI-generated interview questions are displayed
